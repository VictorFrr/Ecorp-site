Guide d'utilisation

Installation

ECORP nécessite Python 3.11 exactement. Une version différente provoquera une erreur au démarrage.

# 1. Téléchargez Python 3.11 # https://www.python.org/downloads/release/python-3110/ # 2. Extrayez le ZIP téléchargé # 3. Ouvrez un terminal dans le dossier # 4. Exécutez le script d'installation setup.bat

Le script setup.bat installe automatiquement : requests, pynput, customtkinter, colorama, psutil, internetspeedtest, clipboard, Pillow, dnspython, bs4, aiohttp, python-nmap, cx_Freeze, pyarmor, nuitka, flet.

Nmap doit être installé séparément depuis nmap.org.

Lancement

# Dans le dossier du projet python main.py

Au démarrage, ECORP demande de choisir la langue :

🌐 Choisissez votre langue / Choose your language (FR/EN) :

Tapez FR pour le français ou EN pour l'anglais. Le choix est modifiable à tout moment via l'option [62].

Modules détaillés

[11] Générateur de mot de passe

Génère un mot de passe aléatoire avec critères configurables (longueur, symboles, chiffres).

[12] Vérificateur de mot de passe

Analyse la robustesse d'un mot de passe saisi (longueur, complexité, entropie).

[14] Mot de passe compromis (HIBP)

Vérifie si un mot de passe a fuité via l'API Have I Been Pwned avec k-anonymat : seuls les 5 premiers caractères du hash SHA-1 sont transmis.

[21] DirBuster

Découverte de répertoires cachés sur un serveur web par force brute de chemins.

[22] Générateur de fausse page HTML

Crée une page de phishing factice à des fins de démonstration et de sensibilisation.

[23] Keylogger

Capture les frappes clavier localement via CustomTkinter. Purement éducatif, aucune donnée n'est transmise.

[31–36] Réseau

Ping IP, scan de ports Nmap, lecture des logs, géolocalisation IP (FAI, pays, ville, ASN), speedtest montant/descendant.

[41] Informations système

Affiche OS, CPU, RAM, stockage, réseau.

[51] Recherche d'utilisateur OSINT

Recherche un pseudonyme sur plusieurs plateformes publiques.

[53] Chiffrement Python

Chiffre un fichier .py pour en protéger le code source.

Quiz interactifs (Flet)

Trois quiz pédagogiques dans une interface graphique Flet :

[13] Quiz Mot de passe [24] Quiz Sécurité (3 niveaux : facile, moyen, difficile) [34] Quiz Réseau

Ces modules nécessitent que flet soit installé. Ils s'ouvrent dans une fenêtre graphique séparée.

Fichier de logs

# Consulter via le menu : [33] # Ou directement : cat logs.txt

Le fichier logs.txt enregistre les actions effectuées lors des sessions. C'est le seul fichier de données créé localement.

FAQ

ECORP est un outil éducatif. Les outils pentest, keylogger ou phishing ne doivent être utilisés que sur vos propres systèmes ou dans un environnement de test autorisé. Toute utilisation sur des systèmes tiers sans autorisation est illégale (art. 323-1 à 323-8 du Code pénal français).
ECORP utilise cx_Freeze pour la compilation en exécutable, qui a des contraintes de compatibilité strictes. Python 3.11 est la version développée et testée. D'autres versions peuvent fonctionner mais cx_Freeze sera désactivé.
Non. ECORP utilise le k-anonymat : seuls les 5 premiers caractères du hash SHA-1 sont transmis. Votre mot de passe réel ne quitte jamais votre machine.
Le keylogger est entièrement local. Il capture les frappes et les affiche dans l'interface graphique, mais ne les envoie à aucun serveur distant.
ECORP a été développé exclusivement sous Windows 10/11. Certains modules utilisent des APIs Windows (winreg, ctypes.windll) et ne fonctionneront pas sur Linux ou macOS.
python-nmap est un wrapper : Nmap doit être installé depuis nmap.org et ajouté au PATH. Relancez ensuite ECORP en tant qu'administrateur.
