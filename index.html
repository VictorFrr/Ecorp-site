Guide d'utilisation

Installation

ECORP nécessite Python 3.11 exactement. Une version différente (3.10, 3.12…) provoquera une erreur au démarrage.

# 1. Téléchargez Python 3.11 # https://www.python.org/downloads/release/python-3110/ # 2. Extrayez le ZIP téléchargé # 3. Ouvrez un terminal dans le dossier extrait # 4. Exécutez le script d'installation setup.bat

Le script setup.bat met à jour pip et installe automatiquement toutes les dépendances : requests, pynput, customtkinter, colorama, psutil, internetspeedtest, clipboard, Pillow, dnspython, bs4, aiohttp, python-nmap, cx_Freeze, pyarmor, nuitka, flet.

Nmap doit être installé séparément depuis nmap.org pour que le scan réseau fonctionne.

Lancement

# Dans le dossier du projet python main.py

Au démarrage, ECORP vous demande de choisir la langue :

🌐 Choisissez votre langue / Choose your language (FR/EN) :

Tapez FR pour le français ou EN pour l'anglais. Le choix peut être modifié à tout moment via l'option [62] dans les paramètres.

Modules détaillés

[11] Générateur de mot de passe

Génère un mot de passe aléatoire avec critères configurables (longueur, symboles, chiffres).

[12] Vérificateur de mot de passe

Analyse la robustesse d'un mot de passe saisi (longueur, complexité, entropie).

[14] Mot de passe compromis

Vérifie si un mot de passe a fuité en interrogeant l'API Have I Been Pwned avec k-anonymat : seuls les 5 premiers caractères du hash SHA-1 sont envoyés.

[21] DirBuster

Découverte de répertoires cachés sur un serveur web par force brute de chemins.

[22] Générateur de fausse page HTML

Crée une page de phishing factice à des fins de démonstration et de sensibilisation.

[23] Keylogger

Capture les frappes clavier localement avec interface graphique (CustomTkinter). Purement éducatif, aucune donnée n'est transmise.

[31] Ping IP

Teste la connectivité vers une adresse IP ou un domaine.

[32] Scan Réseau

Scan de ports via python-nmap sur une plage définie.

[35] IP Lookup

Géolocalise une IP et affiche FAI, pays, ville, ASN.

[36] Speedtest

Mesure le débit montant/descendant de la connexion Internet.

[41] Informations système

Affiche OS, CPU, RAM, stockage, réseau.

[51] Recherche d'utilisateur

OSINT : recherche un pseudonyme sur plusieurs plateformes.

[53] Chiffrement Python

Chiffre un fichier .py pour en protéger le code source.

Quiz interactifs (Flet)

Trois quiz pédagogiques sont disponibles, affichés dans une interface graphique Flet :

[13] Quiz Mot de passe [24] Quiz Sécurité (3 niveaux : facile, moyen, difficile) [34] Quiz Réseau

Ces modules nécessitent que flet soit installé. Ils s'ouvrent dans une fenêtre graphique séparée.

Fichier de logs

# Consulter les logs depuis le menu : [33] # Ou directement : cat logs.txt

Le fichier logs.txt enregistre les actions effectuées lors des sessions. Il est lisible depuis l'option [33] du menu. C'est le seul fichier de données créé localement par ECORP.

FAQ

ECORP est un outil éducatif conçu pour apprendre la cybersécurité dans un cadre pédagogique. Les outils de type pentest, keylogger ou phishing ne doivent être utilisés que sur vos propres systèmes ou dans un environnement de test autorisé. Toute utilisation sur des systèmes tiers sans autorisation explicite est illégale et sanctionnée par le Code pénal français (articles 323-1 et suivants).
ECORP utilise cx_Freeze pour la compilation en exécutable, qui a des contraintes de compatibilité strictes. Python 3.11 est la version sur laquelle le projet a été développé et testé. D'autres versions peuvent fonctionner, mais cx_Freeze sera désactivé et certains comportements peuvent différer.
Non. ECORP utilise la technique du k-anonymat de l'API Have I Been Pwned : seuls les 5 premiers caractères du hash SHA-1 de votre mot de passe sont transmis. Votre mot de passe réel ne quitte jamais votre machine.
Le keylogger est entièrement local. Il capture les frappes et les affiche dans l'interface graphique, mais ne les envoie à aucun serveur distant. Il s'agit d'un outil démonstratif pour comprendre le fonctionnement de ce type de logiciel malveillant.
ECORP a été développé et testé exclusivement sous Windows 10/11. Certains modules utilisent des APIs Windows (winreg, ctypes.windll) et ne fonctionneront pas sur Linux ou macOS. Une adaptation pourrait être envisagée dans une version future.
Téléchargez simplement la dernière version du dépôt GitHub et réexécutez setup.bat pour mettre à jour les dépendances. Le projet est open-source : vous pouvez aussi cloner le dépôt avec Git et effectuer un git pull pour les mises à jour futures.
python-nmap est uniquement un wrapper Python. Vous devez installer Nmap sur votre système depuis nmap.org. Sur Windows, ajoutez Nmap au PATH lors de l'installation. Relancez ensuite ECORP en tant qu'administrateur pour que le scan fonctionne correctement.

protection des données

Informations RGPD

Conformément au Règlement Général sur la Protection des Données (UE) 2016/679 du 27 avril 2016.

Principe général

ECORP est un logiciel qui s'exécute entièrement en local sur votre machine. Il ne dispose d'aucun serveur centralisé, d'aucune base de données distante et d'aucun système d'authentification. Aucune donnée personnelle n'est collectée, stockée ou transmise à l'éditeur du projet.

Base légale des traitements

Les requêtes vers des APIs tierces (HIBP, géolocalisation IP) sont initiées exclusivement à la demande de l'utilisateur et reposent sur le consentement explicite (art. 6.1.a RGPD). Aucun traitement n'est automatique.
